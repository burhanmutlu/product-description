SMOKE S-500PRO+

Automotive Leak Tester

This Automotive Diagnostic Leak Detector is designed for test leaks in vehicle pipe systems. It cloud be used on all cars, motorcycles, snowmobiles, ATV, light trucks, boats,etc.

Introduction

ALT-550 Automotive Leak Tester, is a leakage diagnosis and detection equipment latest developed by our company.
It has most advantages such as building smoke faster, amount of smoke being larger and double test mode. It can effectively help the workman to improve work efficiency and reduce work load. ALT-550 can be used for pipeline leakage detection of motor vehicles such as cars, trucks, heavy duty, buses, snow vehicles, yachts and other motor vehicles, including but not limited to the air intake and exhaust system, fuel system, evap system, cooling system, crankcase, brake pipeline, air conditioning system, cylinder, etc. This machine uses MCU to control working.

Overview

M1-EVAP Mode

It is used for the test of micro-pressure pipeline system, such as EVAP system. When the indicator light is on, it indicates that the current running is EVAP test mode.

M2-TURBO/HD Mode

It is used for the test of normal/high-pressure pipeline system, such as fuel system, radiator, A/C system etc. When the indicator light is on, it indicates that the current running is TURBO/HD test mode.

Smoke

When the indicator light is on, it indicates that the current running is Smoke test mode, to find leak detection by smoke.

Air

When the indicator light is on, it indicates that the current running is Air test mode, to quickly judge whether there is leakage in the system.

M1/M2 key

Function mode select between the EVAP and TURBO/HD.

Smoke/Air key

test mode select between the Smoke and Air.

Set key

set the testing time from 5min to 15min,or from 10min to 30min,cycle selection.
key: Enter, Start, Stop
